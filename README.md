# SDET Foundations

Practice scripts from Phase 1 of my SDET learning roadmap.

## Contents

### Week 1 — Python Basics
- variables.py — data types and variable declarations
- grade_checker.py — if/elif/else conditions
- fizzbuzz.py — loops and conditions combined
- guessing_game.py — while loops and user input
- list_stats.py — functions with return values
- palindrome.py — string manipulation and loops
- vowel_counter.py — string methods
- string_reverser.py — list manipulation
- bubble_sort.py — nested loops and swapping

### Week 2 — Python Applied
- read_csv.py — reading files with open()
- write_csv.py — filtering data and writing output
- error_handling.py — try/except for FileNotFoundError and ValueError
- word_frequency.py — dictionaries and string methods
- merge_dicts.py — merging dictionaries

### Week 3 — TypeScript Basics
- variables.ts — types and type annotations
- employees.ts — interfaces and typed functions
- testResults.ts — interfaces with boolean logic
- asyncBasics.ts — async/await fundamentals
- promiseAll.ts — parallel execution with Promise.all
- bubbleSort.ts — bubble sort in TypeScript

## How to run

### Python scripts
python3 filename.py

### TypeScript scripts
tsx filename.ts
